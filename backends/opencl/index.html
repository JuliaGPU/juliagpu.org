OpenCL

For GPUs that support the vendor-neutral OpenCL standard, the OpenCL.jl package provides a Julia interface to the OpenCL programming model. The package is in the early stages of a rewrite, switching from the traditional OpenCL C-based kernel programming model to a native Julia interface that matches the rest of the JuliaGPU ecosystem.

The package can both be used with system-installed OpenCL runtimes, or with a Julia-distributed OpenCL runtime such as pocl_jll.jl:

julia> using OpenCL

julia> OpenCL.versioninfo()
